# pea-plugin-antd-pro

## Installation

```sh
npm i pea-plugin-antd pea-plugin-antd-pro
```

## Usage

```js
import AntdPlugin from 'pea-plugin-antd'
import AntdProPlugin from 'pea-plugin-antd-pro'

export default {
  plugins: [new AntdPlugin(), new AntdProPlugin()],
}
```
