# pea-plugin-antd-pro

## Installation

```sh
npm i pea-plugin-antd pea-plugin-antd-pro
```

## Usage

```js
import antd from 'pea-plugin-antd'
import antdPro from 'pea-plugin-antd-pro'

export default {
  plugins: [antd(), antdPro()],
}
```
